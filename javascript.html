Welcome to JavaScript

JavaScript is a programming language used to make web pages interactive. It allows you to create dynamic content, handle user interactions, and modify HTML and CSS.

How to Use JavaScript

You can add JavaScript to a web page in three ways:

1. Inline JavaScript

JavaScript can be written directly inside an HTML element.

<button onclick="alert('Hello!')">Click Me</button>

2. Internal JavaScript

JavaScript can be written inside a <script> tag.

<script>console.log("Welcome to JavaScript!") </script>

3. External JavaScript

JavaScript can be placed in a separate file and linked to an HTML document.

<script src="script.js"></script>

JavaScript Code

JavaScript is executed by the browser and can interact with HTML and CSS. It allows you to create dynamic content, handle events, and modify webpage elements.

Running JavaScript

JavaScript code can run directly in the browser. One common way to see it in action is using the alert() function:

alert("This is JavaScript in action!");

JavaScript in the Console

The browser’s console is a great place to test JavaScript code. You can open it by pressing F12 (or Ctrl + Shift + I) and navigating to the Console tab.

console.log("Hello from JavaScript!");

Modifying Web Pages

JavaScript can change the content of an HTML page dynamically.

document.body.innerHTML = "<h1>JavaScript Changed This!</h1>";

Responding to User Actions

JavaScript can run code when a user interacts with the page, like clicking a button.

function sayHello() {alert("Hello, User!");}document.getElementById("myButton").onclick = sayHello;

Example button:

<button id="myButton">Click Me</button>

Data and Variables

In JavaScript, variables are used to store data values that can be referenced and manipulated later. Variables make it easy to store information and use it throughout a program.

Declaring Variables

There are three ways to declare variables in JavaScript:

var age = 25;  // Older way (not recommended)
let name = "John";  // Modern and flexible 
const PI = 3.14;  // Constant value (cannot be changed)

Using Variables

Once a variable is declared, you can use it in your program.

let city = "New York"; 
console.log("I live in " + city); // Output: I live in New York

Data Types in JavaScript

JavaScript variables can store different types of data:

  • String - Text values (e.g., "Hello")
  • Number - Numeric values (e.g., 42)
  • Boolean - True or false values (true or false)
  • Array - A list of values (e.g., ["Apple", "Banana", "Cherry"])
  • Object - A collection of key-value pairs

Example of Different Data Types

let message = "Hello, World!"; // String
let score = 100; // Number
let isLoggedIn = true; // Boolean
let fruits = ["Apple", "Banana", "Cherry"]; // Array
let person = { name: "Alice", age: 30 }; // Object

Updating Variables

Variables declared with let can be updated, but const variables cannot be changed.

let color = "blue";
color = "red"; // This works

const birthYear = 2000;
birthYear = 1999; // Error: Assignment to constant variable

Working with Variables

Variables in JavaScript store values that can change as the program runs. This makes them useful for storing user input, tracking data, and performing calculations.

Updating Variables

Variables declared with let can be reassigned new values.

let age = 25; 
age = 30; 
console.log(age); // Output: 30

Using Variables in Operations

Variables can be used in mathematical operations and combined with text.

let price = 50;
let tax = 5;
let total = price + tax;
console.log("Total price: $" + total); // Output: Total price: $55

Concatenating Strings

Strings (text values) can be combined using the + operator or template literals.

let firstName = "John";
let lastName = "Doe";
let fullName = firstName + " " + lastName;
console.log("Hello, " + fullName + "!"); 

// Using template literals (recommended)
console.log(`Hello, ${fullName}!`);

Incrementing and Decrementing

JavaScript allows you to increase or decrease a variable’s value easily.

let count = 10;
count++; // Adds 1
console.log(count); // Output: 11

count--; // Subtracts 1
console.log(count); // Output: 10

Using const for Constants

If you don’t want a variable to change, use const instead of let.

const birthYear = 2000;
birthYear = 1999; // Error: Assignment to constant variable

The Document Object Model (DOM)

The Document Object Model (DOM) allows JavaScript to interact with and modify HTML elements. It represents the structure of a webpage as a tree of objects that JavaScript can access and change dynamically.

Selecting Elements

You can select and modify elements using JavaScript.

// Change the text of an element with id="demo"
document.getElementById("demo").innerHTML = "Hello, DOM!";

Example HTML element:

<p id="demo">This text will change.</p>

Changing Styles with JavaScript

JavaScript can also modify CSS styles dynamically.

document.getElementById("demo").style.color = "red";

Handling Events

JavaScript can respond to user interactions, like button clicks.

function changeText() {
    document.getElementById("message").innerHTML = "You clicked the button!";
}

document.getElementById("btn").onclick = changeText;

Example HTML:

<p id="message">Click the button to change this text.</p>
<button id="btn">Click Me</button>

Creating and Adding Elements

JavaScript can create new elements and add them to the page.

let newElement = document.createElement("p");
newElement.innerHTML = "This is a new paragraph!";
document.body.appendChild(newElement);

JavaScript Functions

A function is a reusable block of code designed to perform a specific task. Functions help in organizing code, improving readability, and reducing repetition.

Declaring a Function

Functions are defined using the function keyword:

function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alice")); // Output: Hello, Alice!

Calling a Function

To execute a function, you call it by using its name followed by parentheses.

let message = greet("Bob");
console.log(message); // Output: Hello, Bob!

Functions with Multiple Parameters

Functions can take multiple inputs (parameters) and return a result.

function add(a, b) {
    return a + b;
}
console.log(add(5, 3)); // Output: 8

Function Without Parameters

A function doesn’t always need parameters. It can perform an action without taking any input.

function sayHello() {
    console.log("Hello, World!");
}
sayHello(); // Output: Hello, World!

Arrow Functions (Shorter Syntax)

JavaScript provides a shorter way to write functions using arrow syntax:

const multiply = (x, y) => x * y;
console.log(multiply(4, 5)); // Output: 20

Anonymous Functions

Functions can be created without names and assigned to variables.

let divide = function(a, b) {
    return a / b;
};
console.log(divide(10, 2)); // Output: 5

Functions and the DOM

Functions can also interact with the web page.

function changeText() {
    document.getElementById("message").innerHTML = "Text changed!";
}
document.getElementById("btn").onclick = changeText;

Example HTML:

<p id="message">Click the button to change this text.</p>
<button id="btn">Click Me</button>

User Input

JavaScript can take user input using different methods, such as prompt(), confirm(), and input fields from the webpage.

Using the prompt() Function

The prompt() function allows users to enter a value in a popup box.

let userName = prompt("Enter your name:");
        console.log("Welcome, " + userName + "!");

Using confirm() for Yes/No Choices

The confirm() function asks the user a question with "OK" (true) or "Cancel" (false) as responses.

let isConfirmed = confirm("Do you like JavaScript?");
        if (isConfirmed) {
            console.log("Great! Let's learn more.");
        } else {
            console.log("No worries! Maybe you'll like it later.");
        }

Getting User Input from an HTML Form

JavaScript can collect input from form fields and display results on the page.

function showMessage() {
            let name = document.getElementById("nameInput").value;
            document.getElementById("output").innerHTML = "Hello, " + name + "!";
        }

Example HTML:

<input type="text" id="nameInput" placeholder="Enter your name">
        <button onclick="showMessage()">Submit</button>
        <p id="output"></p>

Number and String Operations

JavaScript can perform various mathematical calculations and manipulate text (strings) efficiently.

Number Operations

JavaScript supports basic arithmetic operations:

let num1 = 10, num2 = 5;
          console.log(num1 + num2); // Addition: 15
          console.log(num1 - num2); // Subtraction: 5
          console.log(num1 * num2); // Multiplication: 50
          console.log(num1 / num2); // Division: 2
          console.log(num1 % num2); // Modulus (remainder): 0

Increment and Decrement

Numbers can be increased or decreased easily using ++ and --.

let count = 5;
          count++; // Increments by 1
          console.log(count); // Output: 6
          
          count--; // Decrements by 1
          console.log(count); // Output: 5

String Operations

Strings (text) can be combined using the + operator.

let text = "Hello" + " World!";
          console.log(text); // Output: Hello World!

Using Template Literals

Template literals (` backticks) make it easier to insert variables into strings.

let name = "Alice";
          let greeting = `Hello, ${name}!`;
          console.log(greeting); // Output: Hello, Alice!

Finding String Length

The .length property returns the number of characters in a string.

let word = "JavaScript";
          console.log(word.length); // Output: 10

Changing Case

JavaScript allows you to convert strings to uppercase or lowercase.

let phrase = "JavaScript is fun!";
          console.log(phrase.toUpperCase()); // Output: JAVASCRIPT IS FUN!
          console.log(phrase.toLowerCase()); // Output: javascript is fun!

Extracting Parts of a String

You can extract specific parts of a string using slice() and substring().

let sentence = "Hello, JavaScript!";
          console.log(sentence.slice(7, 17)); // Output: JavaScript
          console.log(sentence.substring(7, 17)); // Output: JavaScript

Comparison Operators

Comparison operators are used to compare values and return a boolean result: true (if the condition is met) or false (if it's not).

Basic Comparison Operators

These operators compare two values:

console.log(10 > 5);  // Greater than: true
console.log(10 < 5);  // Less than: false
console.log(10 >= 10); // Greater than or equal to: true
console.log(5 <= 8);  // Less than or equal to: true

Equality Operators

These check if two values are equal or not.

console.log(10 == "10");  // Loose equality (checks value): true
console.log(10 === "10"); // Strict equality (checks value and type): false
console.log(10 != "10");  // Loose inequality: false
console.log(10 !== "10"); // Strict inequality: true

Strict vs. Loose Comparison

The difference between == and === is that == compares only values, while === checks both values and data types.

console.log(5 == "5");  // true (loose equality, converts string to number)
console.log(5 === "5"); // false (strict equality, different types)

Using Comparison Operators in Conditions

Comparison operators are often used in conditional statements.

let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

Comparing Strings

Strings are compared alphabetically based on Unicode values.

console.log("apple" > "banana"); // false ('a' comes before 'b')
console.log("hello" === "hello"); // true (exact match)

Logical Operators

Logical operators are used to combine multiple conditions and return a boolean result: true or false.

1. AND Operator (&&)

The && (AND) operator returns true only if both conditions are true.

console.log(true && true);   // Output: true
console.log(true && false);  // Output: false
console.log(10 > 5 && 8 < 12); // Output: true

2. OR Operator (||)

The || (OR) operator returns true if at least one of the conditions is true.

console.log(true || false);  // Output: true
console.log(false || false); // Output: false
console.log(10 > 15 || 8 < 12); // Output: true

3. NOT Operator (!)

The ! (NOT) operator reverses a boolean value.

console.log(!true);  // Output: false
console.log(!false); // Output: true
console.log(!(10 > 5)); // Output: false

4. Using Logical Operators in Conditions

Logical operators are commonly used in conditional statements.

let age = 20;
let hasLicense = true;

if (age >= 18 && hasLicense) {
    console.log("You can drive.");
} else {
    console.log("You cannot drive.");
}

5. Combining Multiple Conditions

Logical operators can be combined for complex conditions.

let isSunny = true;
let isWeekend = true;

if (isSunny || isWeekend) {
    console.log("Let's go outside!");
} else {
    console.log("Stay indoors.");
}

6. Short-Circuit Evaluation

JavaScript stops evaluating conditions as soon as the result is determined.

console.log(true || someUndefinedFunction()); // Output: true (second part is never executed)
console.log(false && someUndefinedFunction()); // Output: false (second part is never executed)

Conditional Statements

Conditional statements allow JavaScript to make decisions based on conditions. The main conditional statements are:

1. if Statement

The if statement executes a block of code only if the condition is true.

let temperature = 25;
if (temperature > 20) {
    console.log("It's a warm day.");
}

2. if...else Statement

The if...else statement provides an alternative action if the condition is false.

let isRaining = true;
if (isRaining) {
    console.log("Take an umbrella.");
} else {
    console.log("Enjoy the sunshine!");
}

3. if...else if...else Statement

The else if statement allows multiple conditions to be checked in order.

let score = 85;
if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else {
    console.log("Grade: D or F");
}

4. Nested if Statements

An if statement can be placed inside another if statement.

let age = 18;
let hasID = true;

if (age >= 18) {
    if (hasID) {
        console.log("You can enter.");
    } else {
        console.log("You need an ID.");
    }
} else {
    console.log("You must be 18 or older.");
}

5. Using Logical Operators in Conditions

Logical operators can be used within if statements to check multiple conditions.

let time = 10;
let isWeekend = true;

if (time < 12 && isWeekend) {
    console.log("Good morning! Enjoy your weekend.");
} else {
    console.log("Good day!");
}

6. Ternary Operator

The ternary operator (condition ? trueValue : falseValue) is a shorthand for simple if...else statements.

let age = 18;
let message = (age >= 18) ? "You are an adult." : "You are a minor.";
console.log(message);

While Loops

A while loop executes a block of code repeatedly as long as a specified condition remains true. This is useful when we don’t know in advance how many times the loop should run.

1. Basic while Loop

The loop will keep running as long as the condition is true.

let i = 0;
while (i < 5) {
    console.log(i);
    i++; // Incrementing i to avoid an infinite loop
}

2. Using while for User Input

Loops are often used to repeatedly ask for user input until a valid value is given.

let userInput;
while (!userInput) {
    userInput = prompt("Enter your name:");
}
console.log("Hello, " + userInput + "!");

3. Avoiding Infinite Loops

Make sure the condition eventually becomes false, or the loop will run forever.

// This loop runs forever because the condition never changes
while (true) {
    console.log("This is an infinite loop!");
}

4. Using break to Exit a while Loop

The break statement can be used to exit a loop when a condition is met.

let number = 0;
while (number < 10) {
    console.log(number);
    if (number === 5) {
        break; // Stops the loop when number is 5
    }
    number++;
}

5. Using continue to Skip Iterations

The continue statement skips the current iteration and moves to the next.

let x = 0;
while (x < 10) {
    x++;
    if (x % 2 === 0) {
        continue; // Skips even numbers
    }
    console.log(x);
}

6. do...while Loop

The do...while loop ensures the code runs at least once, even if the condition is false.

let count = 5;
do {
    console.log("Count:", count);
    count++;
} while (count < 5); // Runs once even though condition is false

For Loops

A for loop is used when the number of iterations is known. It consists of three parts:

for (initialization; condition; increment/decrement) {
    // Code to execute in each iteration
}

1. Basic for Loop

This loop runs 5 times, printing numbers from 0 to 4.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

2. Looping Through an Array

A for loop is useful for iterating through arrays.

let colors = ["Red", "Green", "Blue"];
for (let i = 0; i < colors.length; i++) {
    console.log(colors[i]);
}

3. Using break to Exit a Loop

The break statement stops the loop when a condition is met.

for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // Stops the loop when i is 5
    }
    console.log(i);
}

4. Using continue to Skip Iterations

The continue statement skips the current iteration and moves to the next.

for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) {
        continue; // Skips even numbers
    }
    console.log(i);
}

5. Nested for Loops

A for loop can be nested inside another loop to create a grid-like pattern.

for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(`i: ${i}, j: ${j}`);
    }
}

6. Looping Backward

A for loop can count down instead of up.

for (let i = 5; i >= 0; i--) {
    console.log(i);
}

7. Using for...of for Arrays

The for...of loop simplifies iterating over arrays.

let fruits = ["Apple", "Banana", "Cherry"];
for (let fruit of fruits) {
    console.log(fruit);
}
