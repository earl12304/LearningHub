Getting Started with Java

Java is a powerful, object-oriented programming language used for building applications, from desktop software to web applications and mobile apps.

Why Learn Java?

Java is popular due to its platform independence (Write Once, Run Anywhere), strong community support, and versatility.

Installing Java

To start coding in Java, you need to install the Java Development Kit (JDK).

Steps to Install Java:

  1. Download the JDK from the official Oracle website or use OpenJDK.
  2. Install the JDK and set up the environment variables.
  3. Verify the installation by running the command:
java -version

Writing Your First Java Program

Let's write a simple Java program to print "Hello, World!".

Example: Hello World Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Compiling and Running Java Code

After writing the code, follow these steps to run it:

  1. Save the file as HelloWorld.java.
  2. Open a terminal or command prompt and navigate to the file's directory.
  3. Compile the Java program using:
javac HelloWorld.java
  1. Run the compiled program with:
java HelloWorld

Understanding the Code

Breaking down the "Hello, World!" program:

  • public class HelloWorld - Defines a class named HelloWorld.
  • public static void main(String[] args) - The main method where execution starts.
  • System.out.println("Hello, World!"); - Prints text to the console.

Now that you've written your first Java program, you're ready to explore more about Java syntax and concepts!

Using Multiple Statements in Java

In Java, a program consists of multiple statements that are executed sequentially. Each statement performs a specific task, such as declaring variables, performing calculations, or displaying output.

Writing Multiple Statements

Each statement in Java must end with a semicolon ;. Multiple statements are executed in order from top to bottom.

Example: Multiple Print Statements

public class MultipleStatements {
    public static void main(String[] args) {
        System.out.println("This is the first statement.");
        System.out.println("This is the second statement.");
        System.out.println("This is the third statement.");
    }
}

Using Variables in Multiple Statements

We can store values in variables and use them in multiple statements.

Example: Declaring and Using Variables

public class VariableExample {
    public static void main(String[] args) {
        int num1 = 10;
        int num2 = 20;
        int sum = num1 + num2;

        System.out.println("First number: " + num1);
        System.out.println("Second number: " + num2);
        System.out.println("Sum: " + sum);
    }
}

Grouping Statements with Blocks

Multiple statements can be grouped using curly braces {}. This is useful in loops, conditionals, and methods.

Example: Grouping Statements in a Block

public class BlockExample {
    public static void main(String[] args) {
        int number = 15;

        if (number > 10) {
            System.out.println("The number is greater than 10.");
            System.out.println("Performing additional calculations...");
        }

        System.out.println("End of program.");
    }
}

Executing Statements Sequentially

Java executes statements in the order they are written, unless flow control statements (like loops or conditionals) change the execution order.

Example: Sequential Execution

public class SequentialExecution {
    public static void main(String[] args) {
        System.out.println("Step 1: Initializing variables.");
        int a = 5, b = 10;

        System.out.println("Step 2: Performing addition.");
        int sum = a + b;

        System.out.println("Step 3: Displaying result.");
        System.out.println("Sum: " + sum);
    }
}

Understanding Java Program Structure

Every Java program follows a specific structure that includes packages, classes, methods, and statements. Understanding this structure is key to writing clean and efficient code.

Basic Structure of a Java Program

A simple Java program consists of a class, a main method, and executable statements.

Example: Basic Java Program Structure

public class ProgramStructure {
    public static void main(String[] args) {
        System.out.println("Understanding Java program structure!");
    }
}

Components of a Java Program

Let's break down the key components:

  • public class ProgramStructure - Defines a class named ProgramStructure. The filename should match the class name.
  • public static void main(String[] args) - The main method where execution begins.
  • System.out.println() - A statement that prints text to the console.

Classes and Methods

Java programs are built using classes and methods. A class is a blueprint for creating objects, while methods define behaviors.

Example: Class with a Method

public class Greeting {
    // Method to print a greeting message
    public static void sayHello() {
        System.out.println("Hello, welcome to Java!");
    }

    public static void main(String[] args) {
        sayHello(); // Calling the method
    }
}

Understanding Variables in Java

Variables are used to store data in a Java program. Each variable has a type, a name, and a value.

Declaring and Initializing Variables

In Java, you declare a variable by specifying its type and name. You can also assign a value at the time of declaration.

Example: Declaring and Initializing Variables

public class VariablesExample {
    public static void main(String[] args) {
        int age = 25; // Integer variable
        double price = 19.99; // Floating-point variable
        char grade = 'A'; // Character variable
        boolean isJavaFun = true; // Boolean variable

        System.out.println("Age: " + age);
        System.out.println("Price: $" + price);
        System.out.println("Grade: " + grade);
        System.out.println("Is Java fun? " + isJavaFun);
    }
}

Types of Variables in Java

Java has different types of variables based on scope and usage:

  • Local Variables â€“ Declared inside a method or block and accessible only within that block.
  • Instance Variables â€“ Declared inside a class but outside methods, associated with an instance of the class.
  • Static Variables â€“ Declared using the static keyword, shared by all instances of the class.

Example: Local, Instance, and Static Variables

public class VariableTypes {
    int instanceVar = 10; // Instance variable
    static String staticVar = "I am static"; // Static variable

    public void show() {
        int localVar = 5; // Local variable
        System.out.println("Local Variable: " + localVar);
    }

    public static void main(String[] args) {
        VariableTypes obj = new VariableTypes();
        obj.show();
        System.out.println("Instance Variable: " + obj.instanceVar);
        System.out.println("Static Variable: " + staticVar);
    }
}

Variable Naming Rules

When naming variables in Java, follow these rules:

  • Must begin with a letter, $, or _ (cannot start with a number).
  • Case-sensitive (Age and age are different).
  • Cannot use Java reserved keywords.

Type Inference with var

Since Java 10, you can use var to let the compiler infer the type.

Example: Using var

public class VarExample {
    public static void main(String[] args) {
        var number = 42; // Compiler infers it as int
        var text = "Hello Java"; // Compiler infers it as String

        System.out.println("Number: " + number);
        System.out.println("Text: " + text);
    }
}

Performing Math Operations in Java

Java provides various arithmetic operators for performing mathematical calculations such as addition, subtraction, multiplication, and division.

Basic Arithmetic Operations

Java supports the following arithmetic operators:

Increment and Decrement Operators

Generating Random Numbers

Taking User Input in Java

Java allows users to enter data using the Scanner class, which is part of the java.util package. This is useful for interactive programs.

Using the Scanner Class

The Scanner class is used to take different types of user input, such as strings, integers, and floating-point numbers.

Example: Taking User Input

import java.util.Scanner; // Import Scanner class

public class UserInputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in); // Create Scanner object

        System.out.print("Enter your name: ");
        String name = scanner.nextLine(); // Read a string

        System.out.print("Enter your age: ");
        int age = scanner.nextInt(); // Read an integer

        System.out.print("Enter your height (in meters): ");
        double height = scanner.nextDouble(); // Read a decimal number

        System.out.println("Hello, " + name + "! You are " + age + " years old and " + height + "m tall.");

        scanner.close(); // Close scanner to prevent resource leak
    }
}

Scanner Methods for Different Data Types

The Scanner class provides different methods to read various data types:

  • nextLine() - Reads an entire line (string).
  • next() - Reads a single word (string).
  • nextInt() - Reads an integer.
  • nextDouble() - Reads a floating-point number.
  • nextBoolean() - Reads a boolean value (true/false).

Example: Reading Different Data Types

import java.util.Scanner;

public class MultipleInputs {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter a boolean value (true/false): ");
        boolean isJavaFun = scanner.nextBoolean();

        System.out.println("You entered: " + isJavaFun);

        scanner.close();
    }
}

Handling User Input Errors

If the user enters an incorrect data type, an error (exception) may occur. We can prevent this using input validation.

Example: Handling Incorrect Input

import java.util.Scanner;

public class InputValidation {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter an integer: ");

        if (scanner.hasNextInt()) {
            int number = scanner.nextInt();
            System.out.println("You entered: " + number);
        } else {
            System.out.println("Invalid input! Please enter an integer.");
        }

        scanner.close();
    }
}

Using Conditional Statements in Java

Conditional statements allow Java programs to make decisions based on specific conditions. The most commonly used conditional statements are if, else if, else, and switch.

The if Statement

The if statement executes a block of code only if a specified condition is true.

Example: Simple if Statement

public class IfExample {
    public static void main(String[] args) {
        int number = 10;

        if (number > 0) {
            System.out.println("The number is positive.");
        }
    }
}

The if-else Statement

The if-else statement executes one block of code if the condition is true and another block if the condition is false.

Example: if-else Statement

public class IfElseExample {
    public static void main(String[] args) {
        int number = -5;

        if (number > 0) {
            System.out.println("The number is positive.");
        } else {
            System.out.println("The number is negative or zero.");
        }
    }
}

The if-else if-else Statement

The if-else if-else statement checks multiple conditions sequentially.

Example: if-else if-else Statement

public class ElseIfExample {
    public static void main(String[] args) {
        int score = 85;

        if (score >= 90) {
            System.out.println("Grade: A");
        } else if (score >= 80) {
            System.out.println("Grade: B");
        } else if (score >= 70) {
            System.out.println("Grade: C");
        } else {
            System.out.println("Grade: F");
        }
    }
}

The switch Statement

The switch statement is useful when multiple conditions are based on the same variable.

Example: switch Statement

public class SwitchExample {
    public static void main(String[] args) {
        int day = 3;

        switch (day) {
            case 1:
                System.out.println("Monday");
                break;
            case 2:
                System.out.println("Tuesday");
                break;
            case 3:
                System.out.println("Wednesday");
                break;
            case 4:
                System.out.println("Thursday");
                break;
            case 5:
                System.out.println("Friday");
                break;
            default:
                System.out.println("Weekend");
        }
    }
}

Using the Ternary Operator

The ternary operator ?: is a shorthand way to write simple if-else statements.

Example: Ternary Operator

public class TernaryExample {
    public static void main(String[] args) {
        int number = 10;
        String result = (number % 2 == 0) ? "Even" : "Odd";
        System.out.println("The number is " + result);
    }
}

Using While Loops in Java

Loops allow Java programs to execute a block of code multiple times. The while loop continues executing as long as a specified condition remains true.

The While Loop

The while loop repeatedly executes a block of code while a given condition evaluates to true.

Syntax of a While Loop

while (condition) {
    // Code to execute
}

Example: Basic While Loop

public class WhileLoopExample {
    public static void main(String[] args) {
        int count = 1;

        while (count <= 5) {
            System.out.println("Count: " + count);
            count++; // Increment count
        }
    }
}

Using While Loops with User Input

While loops are commonly used when we don't know in advance how many times the loop should run.

Example: Reading Input Until User Enters a Specific Value

import java.util.Scanner;

public class WhileUserInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String input = "";

        while (!input.equalsIgnoreCase("exit")) {
            System.out.print("Enter a word (type 'exit' to stop): ");
            input = scanner.nextLine();
            System.out.println("You entered: " + input);
        }

        scanner.close();
    }
}

The Do-While Loop

The do-while loop is similar to while, but it ensures the code inside the loop executes at least once before checking the condition.

Example: Do-While Loop

public class DoWhileExample {
    public static void main(String[] args) {
        int number = 1;

        do {
            System.out.println("Number: " + number);
            number++;
        } while (number <= 5);
    }
}

Breaking Out of a While Loop

The break statement can be used to exit a loop immediately.

Example: Using Break in a While Loop

public class BreakExample {
    public static void main(String[] args) {
        int count = 1;

        while (count <= 10) {
            if (count == 5) {
                System.out.println("Stopping at count = " + count);
                break;
            }
            System.out.println("Count: " + count);
            count++;
        }
    }
}

Skipping an Iteration with Continue

The continue statement skips the rest of the current iteration and moves to the next one.

Example: Skipping Even Numbers

public class ContinueExample {
    public static void main(String[] args) {
        int number = 0;

        while (number < 10) {
            number++;
            if (number % 2 == 0) {
                continue; // Skip even numbers
            }
            System.out.println("Odd number: " + number);
        }
    }
}

Using For Loops in Java

The for loop in Java is used when the number of iterations is known beforehand. It provides a compact structure for looping.

The Basic For Loop

A for loop consists of three parts: initialization, condition, and increment/decrement.

Syntax of a For Loop

for (initialization; condition; update) {
    // Code to execute
}

Example: Counting from 1 to 5

public class ForLoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Iteration: " + i);
        }
    }
}

Using a For Loop to Iterate Over an Array

For loops are commonly used to process arrays and collections.

Example: Iterating Through an Array

public class ArrayIteration {
    public static void main(String[] args) {
        int[] numbers = {10, 20, 30, 40, 50};

        for (int i = 0; i < numbers.length; i++) {
            System.out.println("Element at index " + i + ": " + numbers[i]);
        }
    }
}

The Enhanced For Loop (For-Each Loop)

The enhanced for loop, also known as a for-each loop, is used to iterate over arrays and collections without using an index.

Example: Using a For-Each Loop

public class ForEachExample {
    public static void main(String[] args) {
        String[] fruits = {"Apple", "Banana", "Cherry"};

        for (String fruit : fruits) {
            System.out.println("Fruit: " + fruit);
        }
    }
}

Nesting For Loops

A for loop can be placed inside another for loop to handle multi-dimensional structures like tables.

Example: Nested For Loop to Print a Multiplication Table

public class NestedForLoop {
    public static void main(String[] args) {
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                System.out.print(i * j + " ");
            }
            System.out.println();
        }
    }
}

Declaring and Initializing Arrays

Arrays can be declared using square brackets [] and initialized with values.

Understanding Methods in Java

A method in Java is a reusable block of code that performs a specific task. Methods help organize code, improve readability, and avoid repetition.

Declaring a Method

A method is defined with a name, a return type, and optional parameters.

Syntax of a Method

returnType methodName(parameters) {
            // Code to execute
        }

Example: Creating and Calling a Method

public class MethodExample {
            // Method definition
            public static void greet() {
                System.out.println("Hello, welcome to Java!");
            }
        
            public static void main(String[] args) {
                // Calling the method
                greet();
            }
        }

Methods with Parameters

Methods can accept parameters to process data.

Example: Method with Parameters

public class MethodWithParams {
            public static void greet(String name) {
                System.out.println("Hello, " + name + "!");
            }
        
            public static void main(String[] args) {
                greet("Alice");
                greet("Bob");
            }
        }

Methods with Return Values

A method can return a value using the return keyword.

Example: Method That Returns a Value

public class ReturnExample {
            public static int add(int a, int b) {
                return a + b;
            }
        
            public static void main(String[] args) {
                int sum = add(5, 10);
                System.out.println("Sum: " + sum);
            }
        }

Method Overloading

Java allows multiple methods with the same name but different parameters.

Example: Overloaded Methods

public class MethodOverloading {
            public static int add(int a, int b) {
                return a + b;
            }
        
            public static double add(double a, double b) {
                return a + b;
            }
        
            public static void main(String[] args) {
                System.out.println("Integer sum: " + add(5, 10));
                System.out.println("Double sum: " + add(5.5, 2.3));
            }
        }

Using Recursion

A method can call itself, a technique known as recursion.

Example: Factorial Using Recursion

public class RecursionExample {
            public static int factorial(int n) {
                if (n == 1) {
                    return 1;
                }
                return n * factorial(n - 1);
            }
        
            public static void main(String[] args) {
                System.out.println("Factorial of 5: " + factorial(5));
            }
        }
