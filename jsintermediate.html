JS and HTML Review

Before diving into intermediate JavaScript, let's review how JavaScript interacts with HTML. JavaScript is used to make web pages interactive by manipulating HTML elements and responding to user actions.

Adding JavaScript to HTML

JavaScript can be added to an HTML document in three ways:

1. Inline JavaScript

You can add JavaScript directly inside an HTML element using the onclick attribute.

<button onclick="alert('Hello, World!')">Click Me</button>

2. Internal JavaScript

JavaScript can be written inside a <script> tag in an HTML file.

<script>
    document.write("Welcome to JavaScript!");
    </script>

3. External JavaScript

JavaScript can be placed in a separate file and linked to the HTML file using the script tag.

<script src="script.js"></script>

Manipulating HTML Elements with JavaScript

JavaScript can access and modify HTML elements using the document object.

Changing Text Content

You can change the text inside an HTML element using document.getElementById().

<p id="demo">Original Text</p>
    <script>
    document.getElementById("demo").innerText = "Text Changed!";
    </script>

Handling User Events

JavaScript can respond to user actions like clicks.

<button id="myButton">Click Me</button>
    <script>
    document.getElementById("myButton").addEventListener("click", function() {
        alert("Button Clicked!");
    });
    </script>

Understanding how JavaScript interacts with HTML is key to writing dynamic web applications. Now, let's move on to more advanced JavaScript concepts!

The Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface that allows JavaScript to interact with and modify HTML and CSS dynamically. It represents the structure of a web page as a tree of objects.

Understanding the DOM Structure

When a web page loads, the browser creates a DOM tree representing the HTML document.


    Document
    ├── html
        ├── head
        ├── body
            ├── h1
            ├── p
            ├── button

Accessing Elements in the DOM

JavaScript provides several methods to select and manipulate elements in the DOM.

1. Selecting Elements

There are different ways to access HTML elements using JavaScript.

Using getElementById()

This method selects an element by its id.

<p id="demo">Hello World!</p>
    <script>
    let element = document.getElementById("demo");
    console.log(element.innerText); // Outputs: Hello World!
    </script>
Using getElementsByClassName()

This method selects elements by their class name.

<p class="text">Paragraph 1</p>
    <p class="text">Paragraph 2</p>
    <script>
    let elements = document.getElementsByClassName("text");
    console.log(elements[0].innerText); // Outputs: Paragraph 1
    </script>
Using querySelector() and querySelectorAll()

These methods allow selecting elements using CSS selectors.

<p class="text">Hello</p>
    <script>
    let element = document.querySelector(".text");
    console.log(element.innerText); // Outputs: Hello
    </script>

Modifying Elements

Once an element is selected, we can modify its content, attributes, or styles.

1. Changing Text Content

Use innerText or innerHTML to change an element’s content.

<p id="demo">Original Text</p>
    <script>
    document.getElementById("demo").innerText = "New Text!";
    </script>

2. Changing Attributes

Use setAttribute() to modify an element's attributes.

<img id="myImage" src="old.jpg" alt="Old Image">
    <script>
    document.getElementById("myImage").setAttribute("src", "new.jpg");
    </script>

3. Changing Styles

Use style property to modify CSS properties.

<p id="demo">Hello</p>
    <script>
    document.getElementById("demo").style.color = "red";
    </script>

Adding and Removing Elements

JavaScript can also add or remove elements dynamically.

Adding Elements

<div id="container"></div>
    <script>
    let newElement = document.createElement("p");
    newElement.innerText = "New Paragraph";
    document.getElementById("container").appendChild(newElement);
    </script>

Removing Elements

<p id="demo">This will be removed</p>
    <script>
    let element = document.getElementById("demo");
    element.remove();
    </script>

The DOM is a powerful tool that allows JavaScript to create dynamic and interactive web pages. Mastering the DOM is essential for building modern web applications.

Events and Objects

JavaScript is an event-driven language, meaning it responds to user interactions like clicks, keypresses, and form submissions. These interactions are handled using JavaScript events. Additionally, JavaScript objects allow us to store and organize data efficiently.

JavaScript Events

Events occur when a user interacts with a webpage. JavaScript can listen for these events and execute code in response.

1. Handling Events

Using the onclick Attribute

One way to handle events is by adding an event directly in HTML.

<button onclick="showMessage()">Click Me</button>
    <script>
    function showMessage() {
        alert("Button Clicked!");
    }
    </script>
Using addEventListener()

A better way to handle events is by using JavaScript’s addEventListener() method.

<button id="myButton">Click Me</button>
    <script>
    document.getElementById("myButton").addEventListener("click", function() {
        alert("Button Clicked!");
    });
    </script>

2. Common JavaScript Events

  • click – When an element is clicked.
  • mouseover – When the mouse hovers over an element.
  • mouseout – When the mouse leaves an element.
  • keydown – When a key is pressed.
  • keyup – When a key is released.
  • submit – When a form is submitted.

JavaScript Objects

Objects are used to store data in key-value pairs. They allow us to group related information together.

1. Creating an Object

<script>
    let person = {
        firstName: "John",
        lastName: "Doe",
        age: 30,
        greet: function() {
            return "Hello, " + this.firstName + "!";
        }
    };
    console.log(person.greet()); // Outputs: Hello, John!
    </script>

2. Accessing Object Properties

Properties can be accessed using dot notation or bracket notation.

<script>
    console.log(person.firstName); // Outputs: John
    console.log(person["lastName"]); // Outputs: Doe
    </script>

3. Adding and Modifying Properties

New properties can be added to objects dynamically.

<script>
    person.job = "Developer";
    person.age = 31;
    console.log(person.job); // Outputs: Developer
    </script>

4. Object Methods

Methods are functions inside objects that perform actions.

<script>
    let car = {
        brand: "Toyota",
        model: "Camry",
        start: function() {
            return this.brand + " " + this.model + " is starting...";
        }
    };
    console.log(car.start()); // Outputs: Toyota Camry is starting...
    </script>

Events and Objects Together

JavaScript objects and events often work together to create interactive applications.

<button id="showInfo">Show Info</button>
    <p id="info">

    <script>
    let user = {
        name: "Alice",
        age: 25,
        getInfo: function() {
            return "Name: " + this.name + ", Age: " + this.age;
        }
    };

    document.getElementById("showInfo").addEventListener("click", function() {
        document.getElementById("info").innerText = user.getInfo();
    });
    </script>

Understanding events and objects is crucial for building interactive applications. Events allow us to respond to user actions, while objects help us structure and manage data efficiently.

JavaScript Timer

JavaScript provides built-in timer functions that allow you to execute code after a delay or repeatedly at a set interval. These functions are useful for animations, countdowns, or scheduling tasks.

setTimeout()

The setTimeout() function executes a function after a specified delay (in milliseconds).

Example: Delayed Message

<script>
    setTimeout(function() {
        console.log("This message appears after 3 seconds");
    }, 3000);
    </script>

In this example, the message will appear in the console after 3 seconds.

setInterval()

The setInterval() function repeatedly executes a function at a specified interval.

Example: Updating Time Every Second

<p id="clock">Time: 0 seconds</p>
    <script>
    let seconds = 0;
    setInterval(function() {
        seconds++;
        document.getElementById("clock").innerText = "Time: " + seconds + " seconds";
    }, 1000);
    </script>

In this example, the page updates the time every second.

clearTimeout() and clearInterval()

You can cancel a timer using clearTimeout() or clearInterval().

Example: Stopping an Interval

<p id="counter">Count: 0</p>
    <button id="stopButton">Stop
    <script>
    let count = 0;
    let counter = setInterval(function() {
        count++;
        document.getElementById("counter").innerText = "Count: " + count;
    }, 1000);

    document.getElementById("stopButton").addEventListener("click", function() {
        clearInterval(counter);
    });
    </script>

Clicking the "Stop" button will stop the counter.

Practical Use: Countdown Timer

Let's create a countdown timer that counts down from 10 to 0.

<p id="countdown">10</p>
    <script>
    let timeLeft = 10;
    let timer = setInterval(function() {
        document.getElementById("countdown").innerText = timeLeft;
        timeLeft--;

        if (timeLeft < 0) {
            clearInterval(timer);
            document.getElementById("countdown").innerText = "Time's up!";
        }
    }, 1000);
    </script>

This script counts down from 10 and stops when it reaches 0.

Timers are essential for animations, user notifications, and scheduling tasks in JavaScript. Mastering them will help you create more interactive and dynamic web applications.

JavaScript Arrays

Arrays are used to store multiple values in a single variable. They allow us to manage lists of items efficiently.

Creating an Array

Arrays can be created using square brackets [] or the Array constructor.

Example: Defining an Array

<script>
    let fruits = ["Apple", "Banana", "Cherry"];
    console.log(fruits); // Outputs: ["Apple", "Banana", "Cherry"]
    </script>

Accessing Array Elements

Each item in an array has an index, starting from 0.

Example: Accessing Elements

<script>
    console.log(fruits[0]); // Outputs: Apple
    console.log(fruits[1]); // Outputs: Banana
    </script>

Modifying Arrays

1. Changing an Element

<script>
    fruits[1] = "Blueberry";
    console.log(fruits); // Outputs: ["Apple", "Blueberry", "Cherry"]
    </script>

2. Adding Elements

Use push() to add an item to the end of the array.

<script>
    fruits.push("Mango");
    console.log(fruits); // Outputs: ["Apple", "Blueberry", "Cherry", "Mango"]
    </script>

Use unshift() to add an item to the beginning of the array.

<script>
    fruits.unshift("Grapes");
    console.log(fruits); // Outputs: ["Grapes", "Apple", "Blueberry", "Cherry", "Mango"]
    </script>

3. Removing Elements

Use pop() to remove the last item.

<script>
    fruits.pop();
    console.log(fruits); // Outputs: ["Grapes", "Apple", "Blueberry", "Cherry"]
    </script>

Use shift() to remove the first item.

<script>
    fruits.shift();
    console.log(fruits); // Outputs: ["Apple", "Blueberry", "Cherry"]
    </script>

Looping Through Arrays

Use a loop to iterate through array elements.

Example: Using forEach()

<script>
    fruits.forEach(function(fruit) {
        console.log(fruit);
    });
    // Outputs:
    // Apple
    // Blueberry
    // Cherry
    </script>

Example: Using a for Loop

<script>
    for (let i = 0; i < fruits.length; i++) {
        console.log(fruits[i]);
    }
    </script>

Array Methods

JavaScript provides several built-in methods to manipulate arrays.

1. map()

Creates a new array by transforming each element.

<script>
    let numbers = [1, 2, 3, 4];
    let squared = numbers.map(num => num * num);
    console.log(squared); // Outputs: [1, 4, 9, 16]
    </script>

2. filter()

Creates a new array with elements that meet a condition.

<script>
    let evenNumbers = numbers.filter(num => num % 2 === 0);
    console.log(evenNumbers); // Outputs: [2, 4]
    </script>

3. reduce()

Reduces an array to a single value.

<script>
    let sum = numbers.reduce((total, num) => total + num, 0);
    console.log(sum); // Outputs: 10
    </script>

Practical Use: To-Do List

Let's create a simple to-do list using arrays.

<input type="text" id="taskInput" placeholder="Enter a task">
    <button onclick="addTask()">Add Task
    <ul id="taskList"></ul>

    <script>
    let tasks = [];

    function addTask() {
        let taskInput = document.getElementById("taskInput");
        let task = taskInput.value;
        if (task) {
            tasks.push(task);
            taskInput.value = "";
            displayTasks();
        }
    }

    function displayTasks() {
        let taskList = document.getElementById("taskList");
        taskList.innerHTML = "";
        tasks.forEach((task, index) => {
            taskList.innerHTML += "<li>" + task + " <button onclick='removeTask(" + index + ")'>Remove</button></li>";
        });
    }

    function removeTask(index) {
        tasks.splice(index, 1);
        displayTasks();
    }
    </script>

Arrays are a powerful way to store and manage data in JavaScript. Understanding how to manipulate them efficiently will help in building dynamic web applications.

Indexing and Traversing Arrays

Arrays in JavaScript store multiple values, and each value is assigned an index. Indexing allows us to access, modify, and remove elements, while traversing helps us go through all elements efficiently.

Array Indexing

Each element in an array is assigned an index, starting from 0.

Example: Accessing Elements by Index

<script>
    let colors = ["Red", "Green", "Blue", "Yellow"];
    console.log(colors[0]); // Outputs: Red
    console.log(colors[2]); // Outputs: Blue
    </script>

Example: Modifying an Element

<script>
    colors[1] = "Purple"; 
    console.log(colors); // Outputs: ["Red", "Purple", "Blue", "Yellow"]
    </script>

Example: Finding the Last Element

<script>
    console.log(colors[colors.length - 1]); // Outputs: Yellow
    </script>

Traversing an Array

Traversing means looping through all the elements of an array to process or display them.

1. Using a for Loop

<script>
    for (let i = 0; i < colors.length; i++) {
        console.log("Color at index " + i + ": " + colors[i]);
    }
    </script>

2. Using forEach() Method

The forEach() method is a simpler way to traverse an array.

<script>
    colors.forEach(function(color, index) {
        console.log("Index " + index + ": " + color);
    });
    </script>

3. Using map() to Transform an Array

The map() method creates a new array by applying a function to each element.

<script>
    let uppercaseColors = colors.map(color => color.toUpperCase());
    console.log(uppercaseColors); // Outputs: ["RED", "PURPLE", "BLUE", "YELLOW"]
    </script>

4. Using filter() to Find Elements

Filters elements based on a condition.

<script>
    let shortColors = colors.filter(color => color.length <= 4);
    console.log(shortColors); // Outputs: ["Red", "Blue"]
    </script>

Practical Use: Displaying an Array in HTML

Let's create a simple list to display an array dynamically.

<ul id="colorList"></ul>
    <script>
    function displayColors() {
        let list = document.getElementById("colorList");
        list.innerHTML = "";
        colors.forEach(color => {
            list.innerHTML += "<li>" + color + "</li>";
        });
    }
    displayColors();
    </script>

Indexing and traversing arrays are fundamental skills in JavaScript, allowing efficient data handling and manipulation.

Arrays and the DOM

JavaScript arrays can be used to dynamically update and manipulate elements in the Document Object Model (DOM). This allows us to display lists, modify content, and create interactive web applications.

Displaying an Array in the DOM

We can use a loop to display an array inside an HTML element like a list.

Example: Rendering a List from an Array

<ul id="fruitList"></ul>
    <script>
    let fruits = ["Apple", "Banana", "Cherry", "Mango"];
    
    function displayFruits() {
        let list = document.getElementById("fruitList");
        list.innerHTML = ""; // Clear existing list
        fruits.forEach(fruit => {
            let li = document.createElement("li");
            li.textContent = fruit;
            list.appendChild(li);
        });
    }

    displayFruits();
    </script>

Adding Elements to the DOM from an Array

We can allow users to add new items to an array and update the DOM accordingly.

Example: Adding a New Item

<input type="text" id="fruitInput" placeholder="Enter a fruit">
    <button onclick="addFruit()">Add Fruit
    <ul id="fruitList"></ul>

    <script>
    function addFruit() {
        let input = document.getElementById("fruitInput");
        let fruit = input.value.trim();
        if (fruit) {
            fruits.push(fruit);
            input.value = "";
            displayFruits();
        }
    }
    </script>

Removing Elements from the DOM

We can allow users to remove items dynamically.

Example: Removing an Item

<ul id="fruitList"></ul>

    <script>
    function displayFruits() {
        let list = document.getElementById("fruitList");
        list.innerHTML = ""; 
        fruits.forEach((fruit, index) => {
            let li = document.createElement("li");
            li.textContent = fruit;
            let btn = document.createElement("button");
            btn.textContent = "Remove";
            btn.onclick = () => removeFruit(index);
            li.appendChild(btn);
            list.appendChild(li);
        });
    }

    function removeFruit(index) {
        fruits.splice(index, 1);
        displayFruits();
    }

    displayFruits();
    </script>

Updating DOM Elements Using Arrays

Arrays can help us dynamically update text and attributes in the DOM.

Example: Cycling Through Colors

<div id="colorBox" style="width: 100px; height: 100px; background-color: red;"></div>
    <button onclick="changeColor()">Change Color

    <script>
    let colors = ["red", "blue", "green", "yellow"];
    let index = 0;

    function changeColor() {
        index = (index + 1) % colors.length;
        document.getElementById("colorBox").style.backgroundColor = colors[index];
    }
    </script>

By combining arrays with the DOM, we can create interactive and dynamic web experiences.

Animating Elements with JavaScript

JavaScript allows you to create animations by changing the properties of an element over time. You can animate elements using the setInterval(), setTimeout(), or CSS animations controlled by JavaScript.

Basic Animation with setInterval()

We can move an element across the screen by gradually changing its position.

Example: Moving a Box

<div id="box" style="width: 50px; height: 50px; background: red; position: absolute; top: 50px; left: 0;"></div>
    <button onclick="moveBox()">Move

    <script>
    function moveBox() {
        let box = document.getElementById("box");
        let position = 0;
        let interval = setInterval(() => {
            if (position >= 300) {
                clearInterval(interval);
            } else {
                position += 5;
                box.style.left = position + "px";
            }
        }, 50);
    }
    </script>

Animating with requestAnimationFrame()

The requestAnimationFrame() method provides a smoother animation compared to setInterval().

Example: Smooth Animation

<div id="circle" style="width: 50px; height: 50px; background: blue; border-radius: 50%; position: absolute; top: 100px; left: 0;"></div>
    <button onclick="animateCircle()">Start Animation

    <script>
    function animateCircle() {
        let circle = document.getElementById("circle");
        let position = 0;

        function step() {
            if (position < 300) {
                position += 5;
                circle.style.left = position + "px";
                requestAnimationFrame(step);
            }
        }

        requestAnimationFrame(step);
    }
    </script>

Fading Effects with JavaScript

We can animate the opacity of an element to create a fade effect.

Example: Fade Out

<div id="fadeBox" style="width: 100px; height: 100px; background: green;"></div>
    <button onclick="fadeOut()">Fade Out

    <script>
    function fadeOut() {
        let box = document.getElementById("fadeBox");
        let opacity = 1;

        let interval = setInterval(() => {
            if (opacity <= 0) {
                clearInterval(interval);
                box.style.display = "none";
            } else {
                opacity -= 0.05;
                box.style.opacity = opacity;
            }
        }, 50);
    }
    </script>

Bouncing Animation

Let's create a bouncing ball effect.

Example: Bouncing Ball

<div id="ball" style="width: 50px; height: 50px; background: orange; border-radius: 50%; position: absolute; top: 200px; left: 100px;"></div>
    <button onclick="bounceBall()">Bounce

    <script>
    function bounceBall() {
        let ball = document.getElementById("ball");
        let position = 0;
        let speed = 5;
        let direction = 1;

        function step() {
            if (position >= 200 || position <= 0) {
                direction *= -1; // Change direction
            }
            position += speed * direction;
            ball.style.top = (200 - position) + "px";
            requestAnimationFrame(step);
        }

        requestAnimationFrame(step);
    }
    </script>

Animations enhance user experience by making interfaces more dynamic and interactive. Mastering JavaScript animations allows you to create engaging web applications.

Animating Elements Based on Events

JavaScript allows you to trigger animations when users interact with elements. You can animate elements in response to events like clicks, hovers, key presses, and more.

Animating on Click

Let's animate a box when a button is clicked.

Example: Moving a Box on Click

<div id="clickBox" style="width: 50px; height: 50px; background: red; position: absolute; top: 50px; left: 0;"></div>
    <button onclick="moveOnClick()">Move Box

    <script>
    function moveOnClick() {
        let box = document.getElementById("clickBox");
        let position = 0;
        let interval = setInterval(() => {
            if (position >= 300) {
                clearInterval(interval);
            } else {
                position += 5;
                box.style.left = position + "px";
            }
        }, 50);
    }
    </script>

Animating on Hover

We can animate an element when the user hovers over it.

Example: Expanding a Box on Hover

<div id="hoverBox" style="width: 100px; height: 100px; background: blue; transition: width 0.5s, height 0.5s;">
    </div>

    <script>
    let hoverBox = document.getElementById("hoverBox");

    hoverBox.addEventListener("mouseover", function() {
        hoverBox.style.width = "150px";
        hoverBox.style.height = "150px";
    });

    hoverBox.addEventListener("mouseout", function() {
        hoverBox.style.width = "100px";
        hoverBox.style.height = "100px";
    });
    </script>

Animating on Key Press

We can move an element when the user presses the arrow keys.

Example: Moving a Box with Arrow Keys

<div id="keyBox" style="width: 50px; height: 50px; background: green; position: absolute; top: 100px; left: 100px;">
    </div>

    <script>
    let keyBox = document.getElementById("keyBox");
    let x = 100;
    let y = 100;

    document.addEventListener("keydown", function(event) {
        if (event.key === "ArrowRight") x += 10;
        if (event.key === "ArrowLeft") x -= 10;
        if (event.key === "ArrowUp") y -= 10;
        if (event.key === "ArrowDown") y += 10;

        keyBox.style.left = x + "px";
        keyBox.style.top = y + "px";
    });
    </script>

Animating on Scroll

We can trigger animations when the user scrolls down the page.

Example: Fading in an Element on Scroll

<div id="scrollBox" style="width: 100px; height: 100px; background: purple; opacity: 0; transition: opacity 1s;">
    </div>

    <script>
    window.addEventListener("scroll", function() {
        let scrollBox = document.getElementById("scrollBox");
        let position = scrollBox.getBoundingClientRect().top;
        let screenHeight = window.innerHeight;

        if (position < screenHeight) {
            scrollBox.style.opacity = 1;
        }
    });
    </script>

Animating elements in response to events creates a more interactive and engaging user experience. These techniques are commonly used in modern web development.
